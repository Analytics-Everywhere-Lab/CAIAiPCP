## Prerequisites

* Your system needs `Miniconda` and `Make` installed in order and run the project artefacts locally (e.g., not using containers).
* If the library scan make target needs to be run, `Trivy` should be installed.


## Local Development Make Targets

* Once you have `Make` and `Miniconda` installed, you can type `make` in a terminal after switching to the project's `assemble` directory. 
* You should see all available, public targets after typing `make` in that directory.
